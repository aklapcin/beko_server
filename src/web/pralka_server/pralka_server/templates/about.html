About Tweeting Pralka

Why

Written by Ola. Published on 10th August, 2013.

The easiest answer is: to answer a joke. I am a girl. A girl who code. Because of that from time to time I hear one joke:

Want your girlfriend to star programming?
Buy her a washing machine.

So I've decided to show that programing a washing machine can be quite interesting task. You only need Raspberry Pi and you can do anything. Make your washing machine tweeting, get monthly statistics about numbers of washing cycles, or send you an email with notification that washing powder is just about to finish. The border is your imagination.

What is also important, it is a great project to learn. You need to get Rasberry Pi running, connect it to machine, process input. You need to design whole inrastructure. Choose which libraries and platforms you will use, maybe try out something new.

And there is one last reason I'ce done this project. Isn't internet connected washing machine just (super-geeky-nerdy-)cool?

Btw, pralka is Polish for washing machine. One of not many words, which are shorter in Polish.

Technology

Code of Tweeting Pralka can be found on github

Project consist of two components(both written in Python):

  • Script running on Raspebbry Pi
  • Web application written in Flask

Script running on Raspberry, is run from cron. Every 5 minuts it takes a photo of washing machine diods. Using OpenCV it detects which diods are on and off. Based on it decided about state of washing machine. Then it calls web application api which is responsible for rest of processing.

Web application collects information about changing state of washing machine. When it detects change of machine state it sends tweets.
